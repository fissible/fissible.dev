## forge:generate

```bash
php artisan forge:generate [options]
```

| Option | Description |
|--------|-------------|
| `--output <path>` | Output file path (default: `openapi.yaml`) |
| `--merge <path>` | Merge with an existing spec rather than overwriting |
| `--routes <pattern>` | Only include routes matching a pattern (e.g. `api/*`) |
