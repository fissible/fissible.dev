---
title: Installation
description: Install shellframe via Homebrew or from source.
---

## Homebrew (recommended)

```bash
brew install fissible/tap/shellframe
```

Verify:

```bash
shellframe --version
```

## From source

```bash
git clone https://github.com/fissible/shellframe
cd shellframe
# Source directly — no build step required
source src/shellframe.sh
```

## Requirements

- bash 3.2 or later (macOS default is 3.2; Linux typically ships 5.x)
- No external dependencies beyond bash and standard POSIX utilities
