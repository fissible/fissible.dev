---
title: Installation
description: Install shellql via Homebrew.
---

```bash
brew install fissible/tap/shellql
```

Verify:
```bash
shql --version
```

## Requirements

- bash 3.2+
- sqlite3 (pre-installed on macOS; `apt install sqlite3` on Debian/Ubuntu)
- shellframe (installed automatically as a Homebrew dependency)
