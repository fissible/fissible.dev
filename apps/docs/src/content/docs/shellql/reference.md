## CLI

```bash
shql <database-file>
shql --connection <name>   # use a sigil connection
```

## Keyboard shortcuts

| Key | Action |
|-----|--------|
| `Tab` / `Shift+Tab` | Move between panels |
| `↑` / `↓` | Navigate list |
| `Enter` | Select / open |
| `/` | Focus search / query input |
| `q` | Quit |
| `?` | Show help |
