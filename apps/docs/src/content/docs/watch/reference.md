## Panels

| Panel | URL | Description |
|-------|-----|-------------|
| Dashboard | `/watch` | Overview of routes, exceptions, drift status |
| Routes | `/watch/routes` | Filterable list of all registered routes |
| Drift | `/watch/drift` | Spec coverage and change detection |
| Spec | `/watch/spec` | View and validate your OpenAPI spec |
| Versions | `/watch/versions` | Semver history from drift |
| Exceptions | `/watch/exceptions` | fault triage UI |
